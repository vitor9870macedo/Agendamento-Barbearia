# Backend (Supabase)

Esta pasta contém documentação, scripts e exemplos de integração com o Supabase.

## Estrutura

- `supabase_schema.sql`: Script para criar as tabelas no Supabase.
- `docs/`: Documentação adicional e exemplos de queries.

## Como usar

1. Crie um projeto no Supabase.
2. Execute o script `supabase_schema.sql` no SQL Editor do Supabase.
3. Configure as permissões e policies conforme necessidade.

## Observações

- O backend é totalmente serverless, utilizando o Supabase como BaaS.
- Toda lógica de negócio sensível deve ser validada também no frontend.
